웹 개발 꿀팁 모음 💡

2025년 1월 14일 | JSY | 10분 읽기
#웹개발 #HTML #CSS #JavaScript

안녕하세요! 오늘은 웹 개발을 하면서 제가 유용하게 사용하고 있는 꿀팁들을 공유해보려고 합니다. 초보자분들부터 중급자분들까지 도움이 될 만한 내용들로 준비했어요!

1. VS Code 단축키 마스터하기 ⌨️

개발 생산성을 높이는 가장 빠른 방법은 에디터를 잘 다루는 것입니다. VS Code의 필수 단축키들을 소개합니다:

💡 자주 쓰는 VS Code 단축키

  • Ctrl + P : 빠른 파일 열기
  • Ctrl + Shift + P : 명령 팔레트
  • Alt + 위/아래 화살표 : 코드 라인 이동
  • Shift + Alt + 위/아래 : 코드 라인 복사
  • Ctrl + D : 같은 단어 선택 (여러 개)
  • Ctrl + / : 주석 토글

2. CSS Flexbox 완벽 정리 🎨

Flexbox는 레이아웃을 만들 때 정말 강력한 도구입니다. 자주 사용하는 패턴들을 정리해봤어요:

가운데 정렬의 모든 것

/* 수평 + 수직 가운데 정렬 */ .center { display: flex; justify-content: center; align-items: center; } /* 수평만 가운데 정렬 */ .horizontal-center { display: flex; justify-content: center; } /* 수직만 가운데 정렬 */ .vertical-center { display: flex; align-items: center; }

반응형 카드 레이아웃

.card-container { display: flex; flex-wrap: wrap; gap: 1rem; } .card { flex: 1 1 300px; /* 최소 300px, 늘어날 수 있음 */ max-width: 400px; }

3. JavaScript 유용한 메서드들 🚀

ES6+ 문법을 활용한 깔끔한 코드 작성법을 소개합니다:

배열 다루기

// 배열에서 중복 제거하기 const uniqueArray = [...new Set(array)]; // 배열 평탄화 (중첩 배열 풀기) const nested = [[1, 2], [3, 4], [5, 6]]; const flat = nested.flat(); // [1, 2, 3, 4, 5, 6] // 조건에 맞는 요소 찾기 const users = [{id: 1, name: 'Alice'}, {id: 2, name: 'Bob'}]; const user = users.find(u => u.id === 2); // 배열 요소 합계 구하기 const sum = numbers.reduce((acc, num) => acc + num, 0);

객체 다루기

// 객체 복사 (얕은 복사) const copy = {...original}; // 객체 병합 const merged = {...obj1, ...obj2}; // 조건부 속성 추가 const user = { name: 'JSY', ...(isAdmin && { role: 'admin' }) }; // 객체에서 특정 키 제거 const {password, ...userWithoutPassword} = user;

4. 성능 최적화 팁 ⚡

🚀 웹 성능 향상 체크리스트

  • 이미지 최적화 (WebP 포맷 사용, lazy loading)
  • CSS/JS 파일 압축 (minify)
  • 브라우저 캐싱 활용
  • 불필요한 리렌더링 방지
  • 비동기 로딩 활용 (async, defer)

5. Git 꿀팁 🔧

Git을 더 효율적으로 사용하는 방법들입니다:

# 실수로 커밋한 내용 되돌리기 git reset --soft HEAD~1 # 특정 파일만 이전 상태로 되돌리기 git checkout -- filename # 커밋 메시지 수정 git commit --amend -m "새로운 메시지" # 브랜치 이름 변경 git branch -m old-name new-name # 깔끔한 로그 보기 git log --oneline --graph --all

6. 디버깅 꿀팁 🐛

버그를 빠르게 찾고 해결하는 방법:

🔍 Console 객체 활용하기

// 테이블 형태로 출력 console.table(array); // 실행 시간 측정 console.time('작업'); // ... 코드 실행 console.timeEnd('작업'); // 그룹화해서 출력 console.group('그룹 이름'); console.log('내용1'); console.log('내용2'); console.groupEnd(); // 조건부 출력 console.assert(condition, '조건이 false일 때 출력');

🎮 MCPO 서버 완벽 가이드

마인크래프트 Pocket Edition 서버 구축부터 운영까지

작성일: 2024년 12월 28일

📋 개요

MCPO(Minecraft Pocket Edition Online) 서버는 모바일 마인크래프트 플레이어들이 함께 플레이할 수 있는 멀티플레이어 서버입니다. 이 가이드에서는 서버 구축부터 관리까지 모든 과정을 다룹니다.

🎯 이 가이드에서 배울 내용:
  • MCPO 서버 설치 및 설정
  • 플러그인 관리 및 최적화
  • 서버 보안 및 백업
  • 플레이어 관리 및 권한 설정

🛠️ 서버 설치

1단계: PocketMine-MP 다운로드

가장 인기 있는 MCPO 서버 소프트웨어인 PocketMine-MP를 설치합니다.

# Linux/macOS wget https://github.com/pmmp/PocketMine-MP/releases/latest/download/PocketMine-MP.phar php PocketMine-MP.phar # Windows # GitHub에서 최신 버전 다운로드 후 실행 php PocketMine-MP.phar

2단계: 초기 설정

서버 첫 실행 시 기본 설정을 진행합니다.

[*] PocketMine-MP 설정 마법사 [*] 서버 언어를 선택하세요 [kor]: kor [*] 서버 이름을 입력하세요: My MCPO Server [*] 서버 포트 [19132]: 19132 [*] 최대 플레이어 수 [20]: 20 [*] 게임모드를 선택하세요 [0]: 0

3단계: server.properties 설정

서버의 세부 설정을 조정합니다.

# server.properties 주요 설정 motd=My Awesome MCPO Server server-name=MCPO Server max-players=20 gamemode=0 difficulty=1 white-list=false online-mode=true spawn-protection=16

서버 내 경제 시스템을 구축합니다.

# 다운로드 및 설치 wget plugin-url/EconomyAPI.phar # plugins 폴더에 복사

# config.yml에서 설정 plotSize: 32 roadWidth: 7 groundHeight: 64

플레이어 계정 보안을 강화합니다.

# 로그인 명령어 /register /login

# 자동 정리 설정 clearInterval: 300 entityLimit: 1000

⚙️ 서버 최적화

메모리 최적화

# PHP 실행 시 메모리 할당 php -d memory_limit=2G PocketMine-MP.phar # pocketmine.yml 설정 settings: memory: global-limit: 2048 main-limit: 512 main-hard-limit: 1024 check-rate: 20 continuous-trigger: true

네트워크 최적화

# server.properties 네트워크 설정 view-distance=8 simulation-distance=4 network-compression-level=6 network-compression-threshold=256

👥 플레이어 관리

기본 관리 명령어

# 플레이어 관리 /kick [reason] # 플레이어 추방 /ban [reason] # 플레이어 밴 /pardon # 밴 해제 /op # 관리자 권한 부여 /deop # 관리자 권한 제거 # 서버 관리 /stop # 서버 종료 /reload # 플러그인 리로드 /save-all # 월드 저장 /tp # 플레이어 텔레포트

권한 그룹 설정

PurePerms 플러그인을 사용한 권한 관리:

# 그룹 생성 /pp group add VIP /pp group add Moderator /pp group add Admin # 권한 부여 /pp group setperm VIP essentials.fly /pp group setperm Moderator essentials.kick /pp group setperm Admin * # 플레이어에게 그룹 할당 /pp user setgroup VIP

🔒 보안 및 백업

⚠️ 보안 주의사항:
  • 정기적으로 서버 소프트웨어 업데이트
  • 강력한 관리자 비밀번호 사용
  • 의심스러운 플레이어 활동 모니터링
  • 백업은 매일 자동으로 수행

자동 백업 스크립트

#!/bin/bash # backup.sh DATE=$(date +%Y%m%d_%H%M%S) BACKUP_DIR="/backup/mcpo" SERVER_DIR="/server/mcpo" # 백업 디렉토리 생성 mkdir -p $BACKUP_DIR # 월드 백업 tar -czf $BACKUP_DIR/worlds_$DATE.tar.gz $SERVER_DIR/worlds/ # 플러그인 데이터 백업 tar -czf $BACKUP_DIR/plugin_data_$DATE.tar.gz $SERVER_DIR/plugin_data/ # 30일 이상 된 백업 삭제 find $BACKUP_DIR -name "*.tar.gz" -mtime +30 -delete echo "백업 완료: $DATE"

🔧 문제 해결

일반적인 문제들

Q: 플레이어가 서버에 접속할 수 없어요
A: 포트 포워딩(19132) 확인, 방화벽 설정, 서버 IP 주소 확인
Q: 서버가 자주 크래시해요
A: 메모리 부족, 플러그인 충돌, PHP 버전 호환성 확인
Q: 플러그인이 작동하지 않아요
A: 서버 버전 호환성, 플러그인 의존성, 설정 파일 확인

# 실시간 로그 확인 tail -f logs/server.log # 오류 로그만 필터링 grep "ERROR" logs/server.log # 특정 플레이어 로그 확인 grep "PlayerName" logs/server.log
